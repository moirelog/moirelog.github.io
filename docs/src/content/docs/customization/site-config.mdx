通过修改 `moire.config.ts` 文件，你可以自定义网站的基本信息。

## 配置文件位置

```
your-repo/
└── moire.config.ts  ← 在这里
```

## 配置参数详解

打开 `moire.config.ts`，你会看到类似这样的配置：

```typescript
export default {
  // 网站标题
  title: "My Moire Blog",
  
  // 作者名称
  author: "Aeris",
  
  // 网站描述
  description: "My personal blog powered by Moire",
  
  // 每页显示的文章数量
  pageSize: 10,
  
  // 主题选择
  theme: "receipt",
  
  // 你所在的时区
  utcOffset: 8
  
}
```

## 参数说明

### title

**类型**：`string`  
**默认值**：`"Moire Blog"`

网站的标题，会显示在：
- 浏览器标签页
- 页面顶部导航栏
- SEO meta 标签

### author

**类型**：`string`  
**默认值**：`"Anonymous"`

作者名称，用于：
- 文章署名
- RSS feed
- SEO meta 标签

### description

**类型**：`string`  
**默认值**：`"A blog powered by Moire"`

网站描述，主要用于 SEO，会出现在：
- Google 搜索结果下方
- 社交媒体卡片
- RSS feed 描述

### pageSize

**类型**：`number`  
**默认值**：`10`

首页每页显示的文章数量。

- 建议 5-20 之间
- 太少需要频繁翻页
- 太多加载慢

### theme

**类型**：`string`  
**可选值**：`"receipt"` | `"cyberpunk"` | `"academic"` | `"pixel"`

主题风格，详见[主题风格](/customization/themes/)章节。

### utcOffset

**类型**：`number`  
**默认值**：`8`

你所在的 UTC 时区，用于计算文章的发布时间。

## 示例配置

一个完整的配置示例：

```typescript
export default {
  title: "莫尔的笔记本",
  author: "Aeris",
  description: "记录技术、生活与思考",
  pageSize: 8,
  theme: "academic",
  avatar: "https://avatars.githubusercontent.com/u/123456",
  utcOffset: 8
}
```

## 应用更改

修改配置后：

<Steps>

1. **提交到 GitHub**（通过 Git 或 GitHub [网页编辑](https://docs.github.com/zh/repositories/working-with-files/managing-files/editing-files)）
2. GitHub Actions 会自动构建
3. 等待 1-2 分钟，刷新网站即可看到更改

</Steps>

:::tip[本地预览]
如果你克隆了仓库到本地，可以运行 `npm run dev` 即时预览配置更改。
:::
