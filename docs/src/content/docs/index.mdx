## 工作原理

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   Apple Notes   │───▶│  iOS Shortcut   │───▶│   GitHub Repo   │
│   写作           │    │  Moire Sync     │    │   存储          │
└─────────────────┘    └─────────────────┘    └─────────────────┘
                                                      │
                                                      ▼
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   Your Website  │◀───│ GitHub Actions  │◀───│   自动触发       │
│   发布           │    │   构建部署       │    │                 │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

## 快速开始

准备好了吗？只需三步即可拥有你的博客：

1. **Fork 仓库** - 复制 Moire 到你的 GitHub
2. **配置 Token** - 生成并配置访问令牌
3. **安装快捷指令** - 下载并配置 iOS 快捷指令
