**Moire** 是一个轻量级的发布工具，它能够将你的 Apple 备忘录同步到个人网站。


## 什么是 Moire？

Moire 让你可以：
- 📝 **在熟悉的环境中写作** - 使用 Apple 原生的备忘录 App
- 🔄 **即时同步** - 关闭备忘录或快捷键即可触发更新
- 🎨 **个性化展示** - 多种精美主题可选
- 🔒 **完全掌控数据** - 所有内容存储在你的 GitHub 仓库

## 核心特性

### 原生体验优先

很多博客工具要求你安装专门的写作软件，学习新的编辑器，或者登录网页后台。Moire 的出发点很简单——让你直接使用 iPhone/iPad/Mac 上原生的备忘录 App。

### 数据所有权

你的笔记不会存储在任何第三方服务器上。所有内容都以 Markdown 格式存储在你自己的 GitHub 仓库中，你可以随时导出、备份或迁移。

### 极低运行成本

Moire 利用 GitHub Actions 和 GitHub Pages（或 Vercel），通常情况下无需支付服务器费用。

## 技术架构

```
用户写笔记
    ↓
iOS 快捷指令 (Moire Sync)
    ↓
GitHub API (上传 Markdown + 图片)
    ↓
GitHub Actions (自动触发构建)
    ↓
SvelteKit 静态站点生成
    ↓
GitHub Pages / Vercel (部署)
```
