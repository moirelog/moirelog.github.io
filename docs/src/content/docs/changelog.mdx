记录 Moire 仓库和快捷指令的更新历史。


## Shortcuts 版本

### v1.2.0 (2026-02-03)

**新功能**：
- 支持自定义 API Base UR
- 添加版本号标识

**改进**：
- 优化图片上传逻辑，提升稳定性
- 修复特殊字符导致的同步失败

**下载**：
- [Moire Config v1.2](/shortcuts/latest/config)
- [Moire Sync v1.2](/shortcuts/latest/sync)

---

### v1.1.0 (2026-02-02)

**新功能**：
- 支持图片上传（每篇 1 张）
- 自动提取笔记标题作为文件名

**改进**：
- 优化网络超时处理
- 改进错误提示信息

---

### v1.0.0 (2026-02-01)

**首次发布**：
- 基础同步功能
- Markdown 文本上传
- Config + Sync 双快捷指令架构

---

## Moire 仓库版本

### v1.0.0 (2026-02-03)

**首次发布**：
- 基础博客功能
- Markdown 渲染
- Receipt / Academic / Pixel 等主题
- GitHub Actions 自动部署

## 即将推出

### 短期计划（1-2 个月）

-  **SEO 优化**：自动生成 sitemap.xml、rss.xml
-  **Markdown 语法**：支持更多 Markdown 语法
-  **笔记卡片**：生成卡片，优化分享展示

### 中期计划（3-6 个月）

-  **评论系统**：集成 Giscus 或其他评论系统
-  **访问链接**：支持单条笔记的访问链接
-  **多图片上传**：支持每篇笔记 3-5 张图片


### 长期愿景

- 🔮 **多来源支持**：支持 Telegram Channel 或其他来源
- 🔮 **在线编辑器**：Web 版笔记编辑
