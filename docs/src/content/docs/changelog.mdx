## Shortcuts 版本

### v1.2.0 (2026-02-03)

**新功能**：
- ✨ 支持自定义 API Base URL（为代理功能铺路）
- ✨ 添加版本号标识

**改进**：
- 🔧 优化图片上传逻辑，提升稳定性
- 🐛 修复特殊字符导致的同步失败

**下载**：
- [Moire Config v1.2](/shortcuts/latest/config)
- [Moire Sync v1.2](/shortcuts/latest/sync)

---

### v1.1.0 (2026-01-20)

**新功能**：
- ✨ 支持图片上传（每篇 1 张）
- ✨ 自动提取笔记标题作为文件名

**改进**：
- 🔧 优化网络超时处理
- 📝 改进错误提示信息

---

### v1.0.0 (2026-01-10)

**首次发布**：
- 🎉 基础同步功能
- 📝 Markdown 文本上传
- ⚙️ Config + Sync 双快捷指令架构

## Moire 仓库版本

### v2.1.0 (2026-02-01)

**新主题**：
- 🎨 新增 **Pixel** 主题（像素游戏风）
- 🎨 新增 **Academic** 主题（学术论文风）

**改进**：
- ⚡ 优化构建速度，减少 30% 构建时间
- 🔧 修复 Safari 浏览器显示问题
- 📱 改进移动端响应式布局

**Breaking Changes**：
- ⚠️ `moire.config.ts` 新增 `theme` 字段（默认 `receipt`）

---

### v2.0.0 (2026-01-15)

**重大更新**：
- 🚀 升级到 SvelteKit 2.0
- 🎨 新增 **Cyberpunk** 主题
- 📦 支持主题切换系统

**新功能**：
- ✨ 支持标签（Tag）功能
- ✨ RSS Feed 自动生成
- ✨ 深色模式支持

**改进**：
- ⚡ 页面加载速度提升 50%
- 🔧 优化图片懒加载

---

### v1.5.0 (2026-01-05)

**新功能**：
- ✨ 支持分页（可配置 `pageSize`）
- ✨ 添加搜索功能（客户端）

**改进**：
- 🔧 修复日期排序问题
- 📝 改进 Markdown 渲染（支持更多语法）

---

### v1.0.0 (2025-12-25)

**首次发布**：
- 🎉 基础博客功能
- 📝 Markdown 渲染
- 🎨 Receipt 默认主题
- 🔄 GitHub Actions 自动部署

## 即将推出

### 短期计划（1-2 个月）

- 🔜 **Bento 主题**：日式便当格子风
- 🔜 **多图片上传**：支持每篇笔记 3-5 张图片
- 🔜 **评论系统**：集成 Giscus

### 中期计划（3-6 个月）

- 🔜 **数学公式**：KaTeX 支持
- 🔜 **Mermaid 图表**：流程图、时序图支持
- 🔜 **全文搜索**：基于 Pagefind

### 长期愿景

- 🔮 **多平台支持**：Android Shortcuts
- 🔮 **在线编辑器**：Web 版笔记编辑
- 🔮 **协作功能**：多作者博客
