在开始安装 Moire 之前，请确保你满足以下条件。

## 网络环境

:::caution[针对中国大陆用户]
请确保你的网络环境能够正常访问 GitHub API。

如果访问受限，请参考 [API 加速](/advanced/api-proxy/) 章节配置代理。
:::

## 账号要求

### GitHub 账号

Moire 依赖 GitHub 存储数据和部署网站，因此你需要一个 GitHub 账号。

如果还没有，请前往 [github.com](https://github.com) 注册。

:::tip[小提示]
GitHub 账号是免费的，注册只需要一个邮箱。
:::

## 设备要求

### iOS / iPadOS

- iOS 15.0 或更高版本
- 需要安装「快捷指令」App（系统自带）

### macOS

- macOS 12 Monterey 或更高版本
- 需要安装「快捷指令」App（系统自带）

## 技术知识

你**不需要**会编程，但以下知识会有帮助：

- ✅ 知道什么是 GitHub 仓库（Repository）
- ✅ 了解 Fork 操作的基本概念
- ✅ 能够阅读简单的英文界面

## 时间预估

整个安装过程大约需要 **10-15 分钟**，包括：

| 步骤 | 时间 |
|------|------|
| Fork 仓库 | 2 分钟 |
| 配置 Actions 和 Pages | 3 分钟 |
| 生成 Token | 3 分钟 |
| 安装快捷指令 | 3 分钟 |
| 配置并测试 | 5 分钟 |
