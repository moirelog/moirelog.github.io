为了让 Moire 知道哪些笔记需要同步，我们需要在备忘录中创建一个专门的文件夹。

## 创建 Moire 文件夹

<Steps>

1. 打开 Apple「备忘录」App

2. 在 **iCloud** 分区下，创建一个新文件夹

3. 将文件夹命名为 **Moire** (或任何你喜欢的名字)

4. 完成！

</Steps>

:::caution[重要：快捷指令绑定文件夹]

快捷指令在分享过程中，所绑定的文件夹以 UUID 形式而不是文件夹名称存在，所以需要重新绑定一次文件夹。
:::

请执行以下操作：

<Steps>

1. 打开「快捷指令」App
2. 找到 **Moire: Sync Notes to Github** 并右键点击编辑
3. 找到步骤「查找所有备忘录」
4. 点击筛选条件中的「文件夹」，并选中你刚才创建的文件夹

</Steps>

## 为什么需要专门的文件夹？

### 更好的组织

专门的文件夹让你可以：
- ✅ 清楚知道哪些内容会公开
- ✅ 更好地管理博客文章
- ✅ 避免误操作

:::tip[最佳实践]
只同步 **Moire** 文件夹中的内容，其他私人笔记保持私密。
:::




## 文件夹位置

确保 Moire 文件夹在 **iCloud** 分区下，而不是"在我的 iPhone 上"。

```
📁 iCloud
  └── 📁 Moire  ← 在这里创建
  
📁 在我的 iPhone 上  ← 不要在这里
```


## 开始创作

创建文件夹后，你就可以在里面写笔记了！

### 写作建议

- 📝 使用 Markdown 语法（粗体、斜体、链接等）
- 🖼️ 可以插入图片（每篇笔记最多 1 张）
- 🏷️ 使用 `#标签` 来分类内容
- ⚠️ 尽量避免在标题或正文头尾出现无意义的空格

## 删除笔记

如果你想从博客中删除一篇已同步的笔记，只需在笔记标题中添加 `[DEL]` 标记，然后重新同步即可。

例如，将标题从：
```
我的第一篇博客
```
改为：
```
[DEL] 我的第一篇博客
```

下次同步时，该笔记将从博客中移除。
