下载快捷指令后，需要编辑 **Moire Config** 来填写你的 GitHub 信息。

## 编辑 Moire Config

<Steps>

1. 打开「快捷指令」App

2. 找到并右键点击 **Moire Config** 并选择「Edit」

3. workflow 中已经有多项配置，找到下面的变量并填入你的信息

</Steps>

## 填写信息


### 1. token

粘贴你在[部署网站](/getting-started/installation/#step-4-生成-token)步骤生成的 Token。

**格式示例**：
```
github_pat_xxxxxxxxxxxxxxxxxxxxxx_xxxxxxxxxxxxxxx
```

### 2. owner (用户名)

输入你的 GitHub 用户名（**不是**邮箱）。

**示例**：
```
aeris
```


### 3. repo (仓库名)

此项一般无需修改，但如果你 Fork 时重命名了仓库，则填写新名称。

**示例**：
```
moire
```

---

:::danger[Token 安全]
- Token 会安全地存储在你的设备上
- 不会上传到任何第三方服务器
- 请妥善保管，不要分享给他人
:::

## 配置示例

完整配置示例：

| 字段 | 值 |
|------|------|
| User | `aeris` |
| Repo | `moire` |
| Token | `github_pat_abc123...` |

填写完成后，这些配置会保存在快捷指令的字典中。

## 常见问题

### 需要在每台设备上都配置吗？

不需要。快捷指令配置**会**在设备间同步，如果你在多台设备（iPhone + iPad）使用 Moire，无需重复配置。
