设置自动化后，你可以在关闭备忘录时或摁下快捷键时自动触发 Moire Sync，无需手动运行。

## iOS 自动化

<Steps>

1. 打开「快捷指令」App

2. 点击底部的 **自动化** 标签

3. 点击右上角 **+** 创建个人自动化

4. 在列表中找到并选择 **App**

5. 点击 **选择**，找到并选择 **备忘录** App

6. 勾选 **已关闭**

7. 勾选 **立即运行**

8. 点击 **下一步**

9. 搜索并选择 **Moire: Sync Notes to Github**

</Steps>

  ![](./ios-automation.png)


:::tip[运行前询问]
如果不关闭"运行前询问"，每次关闭备忘录都会弹出确认对话框，失去了自动化的意义。
:::


## macOS 自动化



  <Steps>
  1. 打开「快捷指令」App
  2. 找到 **Moire: Sync Notes to Github** 右键编辑
  3. 在右侧 tab 中选择符号 `ⓘ` 进入详细信息
  4. 在 **键盘快捷键** 中设置组合键，比如 `^S`
  5. 完成！之后在备忘录中按下快捷键即可同步
  </Steps>

  ![](./mac-automation.png)

## 测试自动化

设置完成后，测试一下：

<Steps>

1. 打开备忘录 App，进入 **Moire** 文件夹

2. 创建一篇新笔记，随便写点内容

3. 关闭备忘录 App 或触发快捷键

4. 你应该看到快捷指令图标闪现，表示正在同步

5. 过几分钟后，访问你的网站，应该能看到新笔记

</Steps>

## 关于同步时机

### 什么时候触发同步？

#### iOS

- ✅ 锁屏 / 退出或关闭备忘录 App
- ❌ 切换到其他笔记（不会触发）

#### macOS

- 快捷键触发

### 同步需要多久？

| 步骤 | 时间 |
|------|------|
| 快捷指令上传 | 3-5 秒 |
| GitHub Actions 构建 | 1-2 分钟 |
| **总计** | **约 2 分钟** |

:::tip[耐心等待]
同步不是即时的，关闭备忘录后稍等 2-3 分钟再刷新网站。
:::
