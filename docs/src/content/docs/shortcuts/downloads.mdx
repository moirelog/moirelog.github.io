Moire 使用两个 iOS 快捷指令来实现同步功能。

## 两个快捷指令

### Moire Config

**作用**：存储你的 GitHub 配置信息（用户名、仓库名、Token）。

**安装顺序**：必须**先**安装这个。

<LinkButton href="/shortcuts/latest/config" variant="primary" icon="download">
  下载 Moire Config
</LinkButton>

### Moire Sync

**作用**：将备忘录内容同步到 GitHub。

**安装顺序**：必须在 Config 之后安装。

<LinkButton href="/shortcuts/latest/sync" variant="primary" icon="download">
  下载 Moire Sync
</LinkButton>

## 依赖关系

:::caution[安装顺序很重要]
必须先安装 **Moire Config**，再安装 **Moire Sync**。

Sync 快捷指令会调用 Config 来获取配置信息，如果没有 Config，Sync 将无法工作。
:::

```
Moire Config (存储配置)
       ↓
Moire Sync (使用配置进行同步)
```

## 安装步骤

<Steps>

1. 在 iPhone/iPad 上点击上方的 **下载 Moire Config** 按钮

2. 系统会询问是否添加快捷指令，点击 **添加快捷指令**

3. 重复步骤 1-2，下载 **Moire Sync**

4. 完成！你应该在「快捷指令」App 中看到两个新快捷指令

</Steps>

## 版本信息

当前版本：**v1.2.0** (2026-02-03)

:::tip[检查更新]
快捷指令内部包含版本号。你可以打开快捷指令查看顶部的注释来确认版本。
:::
