Before you begin installing Moire, please ensure you meet the following requirements.

## Network Environment

:::caution[For Mainland China Users]
Please ensure your network environment can normally access the GitHub API.

If access is restricted, please refer to the [API Acceleration](/en/advanced/api-proxy/) chapter to configure a proxy.
:::

## Account Requirements

### GitHub Account

Moire relies on GitHub to store data and deploy your website, so you will need a GitHub account.

If you don't have one yet, please go to [github.com](https://github.com) to register.

:::tip[Note]
GitHub accounts are free, and you only need an email address to sign up.
:::

## Device Requirements

### iOS

- iOS 16.0 or higher
- Shortcuts app (built-in) must be installed

### macOS

- macOS 13 Ventura or higher
- Shortcuts app (built-in) must be installed

## Technical Knowledge

You **do not** need to know how to code, but the following knowledge will be helpful:

- ✅ Knowing what a GitHub repository is
- ✅ Understanding the basic concept of a Fork
- ✅ Being able to read simple English interfaces

## Estimated Time

The entire installation process takes about **10-15 minutes**, including:

| Step | Time |
|------|------|
| Fork the repository | 2 mins |
| Configure Actions and Pages | 3 mins |
| Generate Token | 3 mins |
| Install Shortcuts | 3 mins |
| Configure and Test | 5 mins |
