## Shortcuts Versions

### v1.2.0 (2026-02-03)

**New Features**:
- Support for custom API Base URL
- Added version number indicator

**Improvements**:
- Optimized image upload logic for better stability
- Fixed sync failures caused by special characters

**Download**:
- [Moire Config v1.2](/en/shortcuts/latest/config)
- [Moire Sync v1.2](/en/shortcuts/latest/sync)

---

### v1.1.0 (2026-02-02)

**New Features**:
- Support for image uploads (1 per post)
- Automatically extract note title as filename

**Improvements**:
- Optimized network timeout handling
- Improved error messages

---

### v1.0.0 (2026-02-01)

**Initial Release**:
- Basic sync functionality
- Markdown text upload
- Dual shortcut architecture (Config + Sync)
