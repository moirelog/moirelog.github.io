## Core Philosophies

### Native Experience First

Many blogging tools require you to install specialized writing software, learn new editors, or log into web backends. Moire is differentâ€”you just use the native Notes app on your iPhone/iPad/Mac.

### Data Ownership

Your notes are not stored on any third-party servers. All content is stored in Markdown format in your own GitHub repository, so you can export, back up, or migrate at any time.

### Zero Server Cost

Moire utilizes GitHub Actions and GitHub Pages (or Vercel), eliminating the need to purchase or maintain servers. It's completely free.
