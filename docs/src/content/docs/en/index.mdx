## How It Works

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   Apple Notes   │───▶│  iOS Shortcut   │───▶│   GitHub Repo   │
│   Writing       │    │  Moire Sync     │    │   Storage       │
└─────────────────┘    └─────────────────┘    └─────────────────┘
                                                      │
                                                      ▼
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   Your Website  │◀───│ GitHub Actions  │◀───│  Auto Trigger   │
│   Publishing    │    │  Build/Deploy   │    │                 │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

## Quick Start

Ready? Get your blog running in three simple steps:

1. **Fork the Repo** - Copy Moire to your GitHub
2. **Configure Token** - Generate and set up your access token
3. **Install Shortcut** - Download and configure the iOS Shortcut
